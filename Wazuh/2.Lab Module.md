* Sigma

Stores backups before upgrades.

### 06_GitHub

# 13. Installing OrbStack

OrbStack provides the virtualization layer for Docker containers.

Responsibilities:

* Creates Linux virtual environment
* Runs Docker Engine
* Manages container networking
* Provides filesystem integration

Verification:

```bash
docker version
```

Expected Output

```text
Client:
 Version: xx.xx

Server:
 Version: xx.xx
```

If Docker returns version information, OrbStack is functioning correctly.

# 18. Certificate Generation

Wazuh uses TLS certificates for encrypted communication.

The following components require certificates:

* Dashboard
* Manager
* Indexer
* Administrator

Generate certificates.

```bash
docker compose -f generate-indexer-certs.yml run --rm generator
```

Purpose

This command generates:

* Root CA
* Manager certificate
* Indexer certificate
* Dashboard certificate
* Administrator certificate

Generated files include:

```text
root-ca.pem
root-ca.key

wazuh.manager.pem
wazuh.manager-key.pem

wazuh.indexer.pem
wazuh.indexer-key.pem

wazuh.dashboard.pem
wazuh.dashboard-key.pem

admin.pem
admin-key.pem
```

These certificates enable secure TLS communication between all Wazuh services.

---

# 19. Validating Certificate Generation

Verify generated certificates.

```bash
find config -name "*.pem"
```

Expected Output

```text
root-ca.pem

wazuh.manager.pem

wazuh.indexer.pem

wazuh.dashboard.pem

admin.pem
```

If any certificate is missing, stop deployment and regenerate certificates before continuing.

# 20. Docker Compose Deployment

Start all services.

```bash
docker compose up -d
```

Expected Output

Docker creates:

* Volumes
* Networks
* Containers

Typical output includes:

```text
Creating volume...

Creating network...

Starting wazuh.indexer

Starting wazuh.manager

Starting wazuh.dashboard
```

Purpose

Docker Compose automatically creates:

* Indexer Container
* Manager Container
* Dashboard Container

and connects them through an isolated Docker network.
